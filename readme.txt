在写函数时的命名约定：
add/create：跳转到添加页，或者打开添加操作的模态窗口
save：执行添加操作
edit：跳转到修改页，或者打开修改操作的模态窗口
update：执行修改操作
get：执行查询操作 find/select/query/...
